# Exclusion Zone Quest Pack

A custom **Project Zomboid** quest pack mod developed for the **Exclusion Zone** dedicated server. This mod introduces immersive questlines, mission progression, objectives, and rewards designed to enhance player engagement, exploration, and roleplay within the server's unique post-apocalyptic world.

---

## Features

* 📜 Custom questlines and story-driven missions
* 🎯 Dynamic objectives and progression tracking
* 🎁 Quest rewards and unlockable content
* 🌎 Server-specific lore and world-building
* 🤝 Multiplayer-friendly quest design
* 🔧 Easily expandable quest framework for future content

---

## About Exclusion Zone

The Exclusion Zone is a dedicated Project Zomboid server focused on creating a rich survival experience through community-driven storytelling, exploration, and roleplay. This quest pack aims to provide survivors with meaningful goals and adventures beyond the standard gameplay loop.

---

## Requirements

* Project Zomboid (Latest Supported Version)
* Dedicated server installation of Exclusion Zone
* Any dependencies required by future quest modules
